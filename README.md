# Monthly Expense Tracker

## Description

Monthly Expense Tracker is a React.js frontend application that helps users manage their monthly expenses.

## Features

- Add Expense
- Edit Expense
- Delete Expense
- Search Expense
- Category Filter
- Monthly Budget
- Dashboard Summary
- Local Storage
- Responsive Design

## Technologies

- React.js
- HTML5
- CSS3
- JavaScript
- Local Storage
